Search

Search the huge internal database by inputting your keywords below with space separated keywords. Eg. "answer love cute have"
You can also search your own files by uploading CSV file with Col1:Date(yyyyMMdd), Col2:Twitter Text format. The first row will be ignored automatically.
WARNING: The file size is limited to 20Mb, or you may get an error from the website.
No matter how large your file might be, you will only get a dataset with size of 200000.
Also, please be patient as the dataset is very huge and searching will take you a long time.
